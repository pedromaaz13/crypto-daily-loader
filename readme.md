# Cripto Dashboard (Supabase + Python + Streamlit)

Este proyecto:

- Extrae datos de CoinGecko
- Almacena los datos en Supabase
- Visualiza los datos con Streamlit

1. Clona el repo

5. Ejecuta `data_loader.py` para cargar datos
